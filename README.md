## About The Project

This project is a Dinosaur Game,also known as the T-Rex Game. The project includes:
* GAME.EXE
* GAME.OBJ
* coalFile
* game.asm

### Built With

* [Assembly Language x86]
* [DOSBOX]

### Prerequisites
* DOSBOX
* Masm

<!-- USAGE EXAMPLES -->
## Usage

For Code:
* Open the coalFile for code.

For Testing:
* Open the GAME.EXE in DOSBOX to run the game.
